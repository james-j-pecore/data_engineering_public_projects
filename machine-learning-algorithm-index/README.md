# Machine Learning Algorithm Index

This repository is a personal index of common machine learning algorithms. Each algorithm has a short, consistent notebook-style entry you can use as study notes or as a quick reference.

Purpose
- Provide concise descriptions, intuition, math, pros/cons, hyperparameters, and example code for common ML algorithms.

How to use
- Read entries to refresh on concepts.
- Use the example code templates to run small experiments.

Per-algorithm template

For each algorithm below, fill in the sections consistently:

- **Overview:** Short description in one or two sentences.
- **Intuition:** Non-technical explanation of how it works.
- **Mathematical formulation:** Key equations or objective.
- **Typical hyperparameters:** Common hyperparameters and reasonable defaults.
- **Advantages:** When to use it.
- **Limitations:** Known failure modes and caveats.
- **Simple example:** Small code snippet or pseudocode.
- **Resources:** Links to papers, tutorials, and scikit-learn docs.
